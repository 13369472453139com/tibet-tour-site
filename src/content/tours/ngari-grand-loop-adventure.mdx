# 阿里大环线探险（13天12晚）

## 行程概览
从拉萨出发，经羊卓雍措、日喀则、珠峰大本营、萨嘎、普兰、札达、狮泉河、改则、尼玛、纳木错，最终返回拉萨。全程约4000公里，涵盖西藏最壮丽的自然与人文景观。

## 详细行程

**D1：拉萨集合**
- 适应海拔（3650米）
- 自由活动：八廓街、大昭寺
- 住宿：拉萨四星酒店

**D2：拉萨 → 羊卓雍措 → 日喀则**
- 车程：约360公里，6小时
- 亮点：羊卓雍措（观景台）、卡若拉冰川、江孜宗山古堡
- 住宿：日喀则三星酒店

**D3：日喀则 → 珠峰大本营**
- 车程：约340公里，7小时
- 亮点：扎什伦布寺、嘉措拉山口、珠峰108拐、绒布寺
- 住宿：绒布寺帐篷营地（海拔5200米）

**D4：珠峰 → 萨嘎**
- 车程：约300公里，6小时
- 亮点：珠峰日出、佩枯措、希夏邦马峰
- 住宿：萨嘎宾馆

**D5：萨嘎 → 普兰**
- 车程：约550公里，8小时
- 亮点：玛旁雍措（圣湖）、拉昂措（鬼湖）、冈仁波齐远眺
- 住宿：普兰酒店

**D6：普兰 → 札达**
- 车程：约330公里，6小时
- 亮点：古格王朝遗址、托林寺、札达土林
- 住宿：札达宾馆

**D7：札达 → 狮泉河**
- 车程：约250公里，5小时
- 亮点：霞义沟土林、班公措（边境湖）
- 住宿：狮泉河酒店

**D8：狮泉河 → 改则**
- 车程：约500公里，8小时
- 亮点：羌塘草原、野生动物（藏羚羊、野牦牛）
- 住宿：改则招待所

**D9：改则 → 尼玛**
- 车程：约450公里，7小时
- 亮点：洞措、当惹雍措、文布南村
- 住宿：尼玛宾馆

**D10：尼玛 → 班戈**
- 车程：约350公里，6小时
- 亮点：色林措（西藏最大咸水湖）、错鄂鸟岛
- 住宿：班戈酒店

**D11：班戈 → 纳木错**
- 车程：约260公里，5小时
- 亮点：纳木错圣象天门、扎西半岛
- 住宿：纳木错板房

**D12：纳木错 → 拉萨**
- 车程：约220公里，4小时
- 亮点：纳木错日出、那根拉山口
- 住宿：拉萨四星酒店

**D13：拉萨送机**
- 结束行程
