# 珠峰东坡徒步·嘎玛沟秘境

## 线路概览
- **难度等级**：★★★★☆（高海拔专业级徒步）
- **最佳季节**：4-5月、9-10月
- **最高海拔**：5350米（朗玛拉垭口）
- **徒步里程**：约120公里
- **住宿方式**：帐篷露营+村落民宿

## 行程详情

### 第1天：拉萨集合（3650m）
- 抵达拉萨，适应高原环境
- 下午自由活动（推荐：八廓街、布达拉宫广场）
- 晚上行前说明会，检查装备

### 第2天：拉萨 → 日喀则（3850m）
- 沿雅鲁藏布江行驶，车程约6小时
- 参观扎什伦布寺
- 宿：日喀则酒店

### 第3天：日喀则 → 曲当乡（3600m）
- 经拉孜、定日，车程约8小时
- 沿途欣赏喜马拉雅山脉
- 宿：曲当乡客栈

### 第4天：曲当乡 → 优帕村 → 晓乌措（4600m）
- 徒步约6小时，海拔爬升1000米
- 途径杜鹃林与溪流
- 宿：晓乌措营地

### 第5天：晓乌措 → 卓湘（4000m）
- 翻越晓乌拉垭口（4900m）
- 下降至嘎玛沟河谷，原始森林景观
- 徒步约7小时
- 宿：卓湘营地

### 第6天：卓湘 → 汤湘（4500m）
- 穿越密林与草甸
- 远眺珠穆朗卓峰
- 徒步约6小时
- 宿：汤湘营地

### 第7天：汤湘 → 巴当 → 俄嘎（4700m）
- 沿山腰横切，冰川融水溪流
- 近距离观赏珠峰东坡巨大冰壁
- 徒步约5小时
- 宿：俄嘎营地

### 第8天：俄嘎 → 珠峰东坡大本营（5300m）
- 轻装前进，经碎石坡与冰川末端
- 抵达珠峰东坡大本营，直面世界之巅
- 返回俄嘎营地
- 徒步约8小时

### 第9天：俄嘎 → 措学仁玛（5000m）
- 返回汤湘后转向措学仁玛
- 三湖连珠，雪山倒影绝佳拍摄点
- 徒步约7小时
- 宿：措学仁玛营地

### 第10天：措学仁玛 → 朗玛拉垭口（5350m） → 拉则措 → 曲当乡
- 翻越全程最高点朗玛拉垭口
- 下降至拉则措，返回曲当乡
- 徒步约9小时
- 宿：曲当乡客栈

### 第11天：曲当乡 → 日喀则
- 乘车返回日喀则
- 庆功晚宴
- 宿：日喀则酒店

### 第12天：日喀则 → 拉萨
- 返回拉萨，结束行程

- 全程帐篷、防潮垫、公用炊具
- 行程所列住宿（拉萨/日喀则为准四星酒店）

- 进山费、环保费、门票
- 公共装备（卫星电话、氧气瓶、急救包）

## 费用不含
- 往返拉萨大交通
- 个人登山装备（睡袋、登山杖等可租赁）
- 单房差
- 个人消费及酒水
- 因不可抗力产生的额外费用

## 注意事项
1. 需有海拔5000米以上徒步经验
2. 提前15天服用红景天，建议购买高原保险
3. 全程无信号（卫星电话仅应急）
4. 尊重当地藏族风俗，不随意丢弃垃圾
5. 听从向导安排，严禁私自离队

## 装备建议
- 羽绒睡袋（舒适温标-10℃）
- 冲锋衣裤、抓绒衣、羽绒服
- 高帮防水徒步鞋、雪套、冰爪
- 墨镜、防晒霜、唇膏、头灯
- 个人药品、保温杯（1L以上）
